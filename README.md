# AI-Powered Microlearning Platform (Frontend)

This project is a modern microlearning platform interface built using Vite, Tailwind CSS, and TypeScript. It provides a mobile-first user experience inspired by reels-style learning applications.

---

## Features

### Learning Interface

* Reels-style course browsing
* Course progress tracking UI
* Module-based learning structure
* Clean and responsive mobile-first design

### UI Components

* Dashboard with analytics cards
* Course detail page with module list
* Creator dashboard layout
* Interactive UI elements and transitions

---

## Tech Stack

* Framework: Vite + TypeScript
* Styling: Tailwind CSS
* Build Tool: Vite
* Package Manager: npm / bun

---

## Project Structure

```
/public        Static assets
/src           Application source code
index.html     Main HTML entry point
package.json   Project dependencies
vite.config.ts Vite configuration
tailwind.config.ts Tailwind configuration
```

## Installation and Setup

1. Clone the repository:

```
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

2. Install dependencies:

```
npm install
```

3. Run development server:

```
npm run dev
```

4. Open in browser:

```
http://localhost:5173
```

---

## Build for Production

```
npm run build
```
